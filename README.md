# HexPath Explorer

HexPath is a hexagonal map pathfinding game designed to introduce children to path optimization and least-energy-path concepts. With a fun, colorful, and simple user interface, it makes learning about algorithms and energy efficiency engaging and accessible.

## Features

- **Hexagonal Grid:** Explore a world built on hexes.
- **Path Optimization:** Find the most efficient route between points.
- **Educational Concepts:** Learn about least-energy paths through play.
- **Colorful UI:** Designed for children with a vibrant and intuitive interface.

## Tech Stack

- **Framework:** [Astro](https://astro.build/)
- **UI Library:** [React](https://reactjs.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **AI:** [Google Gemini API](https://ai.google.dev/)

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in `.env.local` to your Gemini API key.
3. Run the app:
   `npm run dev`

## Deployment

This app is prepared for deployment on GitHub Pages.
